# Smart TV Keyboard

Virtual keyboard optimized for Smart TV with spatial navigation. Pure JavaScript, no dependencies.

## Features

- **Spatial Navigation** - Built for TV remotes, use arrow keys to navigate between keys
- **Vanilla JS** - No dependencies, no frameworks
- **Custom Layouts** - Define your own key layouts easily
- **Display Modes** - Inline or modal display options
- **Disable Keys** - Programmatically disable specific keys
- **TV Optimized** - Large touch targets, high contrast, clear focus states

## Installation

```bash
npm install smart-tv-keyboard
```
